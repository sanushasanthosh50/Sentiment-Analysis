\# Sentiment Analysis Project



This is a machine learning project that predicts whether a text review is positive or negative.



\## Features

\- Trained on IMDB movie review dataset

\- Uses TF-IDF vectorization

\- Machine Learning model for classification

\- Simple Flask web interface



\## Tech Stack

\- Python

\- Scikit-learn

\- Flask

\- HTML/CSS

\- Pandas



\## How it works

1\. User enters a review

2\. Text is cleaned and vectorized

3\. Model predicts sentiment (Positive / Negative)



\## How to run locally

```bash

pip install -r requirements.txt

python app.py
